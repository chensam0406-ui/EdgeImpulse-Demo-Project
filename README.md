* 專題目的: 開發一個有效的影像偵測腳本，訓練模型即時辨別影像，提升模型對三個手勢(剪刀，石頭，布)的準確度辨別，使得我們可以一直贏。

* 使用技術: 程式語言，影像處理，資料集

* 應用場景: 教育娛樂領域，人機互動。
