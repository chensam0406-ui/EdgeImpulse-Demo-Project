# paper scissors stone

## 專題簡介

* 專題目的: 本專案的目的在於結合機器學習與自動化技術，實作一套具備即時互動性的手勢辨識遊戲系統，透過流程自動化提升開發效率，並將影像辨識應用延伸為具娛樂性的實際互動。

* 使用技術: 程式語言，影像處理，資料收集

* 應用場景: 教育娛樂領域，人機互動。
